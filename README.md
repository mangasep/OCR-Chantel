# OCR-Chantel
Sample apps document extractor to text using API opencv

### Prerequisites

* JDK 1.8
* Android SDK
  - Android NDK

### Build

* Clone the repo or download this file
* Build it using Android Studio or gradle script

### Feature

* Camera text recognition
